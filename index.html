Entity's End

Engine: Unreal Engine 5

1 Month Game Jam

Role: Gameplay Programmer, Level Designer, Project Manager

Entity's End is my first 3D game with Unreal Engine 5. It's a SlenderMan-like game that was create me and three other students. Unlike prior projects, this game was created in only one month due to it being for the itch.io Game Jam. It was also our semester project. In this game, the goal is to dig up and collect all the artifacts before the Entity finds you. In this project, I also took the role of the project manager by organizing sprints and meetings. I also created the entire level and coded, with C++ and Blueprints, a lot of the gameplay mechanics. Play the game HERE. View the github repository HERE.

Remember Something?

Engine: Unity

Timeframe: 3 months

Role: Gameplay Programmer, Level Designer, Project Manager
https://github.com/Ironoli259/RememberSomethingFinal

Remember Something is the first full game I created where I was part of a group (group of 3 people). This 3d Unity game was developed throughout the semester and is the game I'm most proud of. During this project, I took the lead as the Project Manager by creating/managing sprints and organizing a scrum meeting every 2 weeks. I also created every map except for one, which was obtained from the Unity asset store. I programmed quite a bit of the game. The game was version controlled using PlasticSCM. Rat Attack

Game Engine: Unity

Timeframe: 1.5 Months

Created Entirely by Me

This is the first game I made in Unity. My teacher tasked me to create a Vampire Survivor clone. This was my final project assignment that I worked on during the entire semester. You collect XP and gold from the enemies you slay. You can use the XP to unlock/upgrade powers. Use the gold to either unlock new playable character or upgrade your player's stats. It is actually finished though it is missing a bit of polishing. During the final stages of this project, I lost both my dad and my dog and lost all motivation to keep working on it.
